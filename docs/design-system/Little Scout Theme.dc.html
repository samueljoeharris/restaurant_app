The Whimsical Theme Kit
A warm, friendly refresh for the kid-food-speed app parents actually open twice. Subtle whimsy on a clean, trustworthy base — the friendly TTF speed tiers stay; the palette turns light & sunny (sky blue · mango · warm ivory), corners soften, and a little scout spirit threads through. Everything below maps to design/tokens.json so a refactor is a 1-to-1 swap.

✓ Do
Round generously · let ivory paper breathe · one playful accent per surface · use emoji as friendly wayfinding only where the brand already does · keep the TTF number the loudest thing on a detail page.
✗ Don't
No gradients-on-everything · no cool gray surfaces (warm ivory only) · don't stack accents (sky-blue AND mango fighting on one card) · no sharp corners · don't let whimsy bury the data parents came for.
