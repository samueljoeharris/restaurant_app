The app's mobile bottom navigation. Emoji glyphs are the brand's wayfinding icons.

```jsx
<BottomNav items={[
  { icon: "🧭", label: "Feed", active: true },
  { icon: "🗺️", label: "Explore" },
  { icon: "💛", label: "Saved" },
  { icon: "🙂", label: "You" },
]} />
```

Exactly one `active`. Standard set is Feed / Explore / Saved / You.
