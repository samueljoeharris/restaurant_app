Surfaces one change on a followed restaurant in the Feed/Home. Tier dot + name + time (or "New" pill).

```jsx
<UpdateCard name="Maria's Cantina" tier="fast" time="2h">
  Kid food now <b>6 min</b> (was 9) 🎉
</UpdateCard>
<UpdateCard name="Tello's Pizza" tier="ok" isNew>New note added.</UpdateCard>
```

Keep body to one friendly line; celebratory emoji (🎉) only on genuine improvements.
