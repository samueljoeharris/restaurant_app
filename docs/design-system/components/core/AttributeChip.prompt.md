Pill for parent attributes and filters (high chairs, changing table, "TTF < 10"). Selectable.

```jsx
<AttributeChip label="High chairs" icon="🪑" />
<AttributeChip label="TTF < 10" active />
<AttributeChip label="more" dashed />
```

Default = white with ivory border; `active` = sky fill; `dashed` = the "+ add" affordance. Lay out in a `flex` row with `gap`.
