How to ship this for real
Three ways to get a custom basemap
A styled basemap is the one piece you can't do in plain CSS in production — it comes from the map provider. The pins, clusters, callouts & controls above are all your own DOM/markers on top, so those port directly. Pick a basemap path:
1Stay on Google
Lowest lift — you already use MapKit/Google. Create a cloud-based Map Style in Google Cloud Console, attach it to a Map ID, and pass that ID to the map. Set land/water/park colors to our tokens and turn POI & transit labels off. (Legacy inline JSON styling is deprecated for new maps — use the Map ID.)
2Mapbox Studio
Most design control. Fork a light base in Mapbox Studio, recolor land/water/parks/roads to the tokens, hide POI layers, publish, and load the style URL via Mapbox GL JS / the iOS SDK. Custom marker images = our teardrop pins.
3MapLibre + MapTiler
Open-source, no per-load vendor lock. MapLibre GL renders a free vector style (e.g. MapTiler) you edit as JSON to match the tokens. Same marker/cluster API as Mapbox. Good if map cost or independence matters.
