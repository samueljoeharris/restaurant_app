1bSystem decisions — resolve in the DS
MEDPaper-dot texture on the whole page body DS says dot-grid is for large empty areas, sparingly. Web applies it to every page. Decide: keep (and codify in DS) or strip to white.
MEDOff-palette map-pin colors --color-pin-ratings #9B6FD9 (purple) and --color-pin-notes #5BA8D6 exist only in web. Purple is outside the Bluebird palette — either add a sanctioned "signal" ramp to the DS or retire the pin classes.
MEDDark mode: class vs media query Web ships a manual .dark toggle (useTheme); DS only ships prefers-color-scheme. Adopt the class strategy into the DS tokens so both stay in sync.
LOWRadius scale mismatch Web: 10/14/20/28. DS: buttons 14, cards 18, pills full. Cards render at 20px on web. Pick one scale and regenerate.
LOWIcons: SVG sprite vs emoji Web uses public/icons.svg stroke icons; DS mandates emoji wayfinding with Lucide as the sanctioned fallback. Confirm the sprite is Lucide-derived and note it in the DS iconography card.
LOWMap park green Web #A8C9A0 vs DS sage #DCEAD2; land uses cream page bg vs DS warm ivory #F6EFE1.
